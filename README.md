# 🐍 Игра "Змейка"

Классическая игра на Python.

## Как запустить на телефоне
1. Установите **Pydroid 3** из Play Market
2. Скачайте файл змейка.py
3. Откройте в Pydroid и нажмите Запуск

## Как запустить на компьютере
1. Установите Python с python.org
2. Скачайте файл змейка.py
3. В терминале введите: python змейка.py
